The Client

The client contains everything that will be visible in the browser window like the editor, the PDF preview and the download options. It depends on angular.js, textAngular and the Mozilla pdf viewer on the programming side and on less and bootstrap for the styles.

Filestructure

client/
  _less/              # addintions to the bootstrap template
    index.less        # this file is compiled and included
  css/                # contains the fonts and the compiled index.less

  js/                 # the angular setup
    config/
      routes.js       # utilizes $routeProvider to set the routes
    controllers/
      editor/         # the controllers for the editor interface
    directives/
      pdf-preview/    # the multi file pdf-preview directive
    app.js

  templates/
    directives/
      pdf-preview/    # template for the multi file pdf-preview
    pages/
      editor/         # template for the editor
    index.html        # the starting point for the angular app
