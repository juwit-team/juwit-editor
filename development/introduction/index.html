Introduction to the different parts

The Editor has three main components that are basicly interchangable with other similar implementations.

Warning

The API for all components is not yet ready and it is most likely that some interfaces will change in the future.

Concept of these three components

Client

The client contains everything that will be visible in the browser window like the editor and the

Parser

The parser handles the conversion from (valid) html to (valid) latex. It utilizes the the latex-sanitizer and html-parser2.

Server

The server gets the requests, starts the parser and basicly delivers the client folder to the browser.
