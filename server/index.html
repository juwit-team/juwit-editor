The Server

The server gets the requests, starts the parser and basicly delivers the client folder to the browser.

Filestructure

server/
  server.js                 # starting point for the server

  _texFiles/                # contains everything latex related
    company/                # this is an example company with it's documents
      logo.png              # the company logo for the letter head
      document1/            # an example document
        uploads/            # uploaded configs end up here
        xyz.cls             # latex class file for type `xyz` ex. letter
        document1.tex       # current latex file that will be compiled

  config/
    express.js              # express.js config

  models/                 
    compiler.js             # concats the .tex file, starts pdflatex.js
    formletter-compiler.js  # like compiler + multi recepients handling
    pdflatex.js             # wrapper for commandline `pdflatex`

  routes/
    compile.js              # `/:group/:document/compile`
    updateCSV.js            # `/:group/:document/updateCSV`
