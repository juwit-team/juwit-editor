Getting started

Install dependencies

The following things are needed to properly run the server

production

  • node.js & npm
  • latex
  • maybe bower (needs further testing)

development

  • less
  • mkdocs

Install the editor

Download the latest version from the repository. Then go to the folder you copied it to and run npm start. This will install all the dependencies and start your server at localhost:1338

Run the tests

After you installed everything just use npm test to start the test suite.
