Just Write It - Editor

The almost what you see is what you get latex editor.

What is it?

An editor for writing text that compiles beautifully to latex and therefore always looks professional in the generated PDF.

Who is it for?

The main target are users that are not familiar with markup languages in general and latex in specific. The goal is to enable those users to write professinal company letters without any learning curve.

How to get started

Note

At this point it is only tested on lokal machines working as server and client

Install on Linux, Mac and Windows

  1. install latex
  2. install node.js
  3. install bower with npm install -g bower
  4. download the juwit-editor and unzip it

Additional dependencies

  • to edit the styles you should npm install -g less
  • to edit this documentation you will need python & mkdocs

Start the server

  1. cd path/to/juwit/editor
  2. npm start
  3. open browser with http://localhost:1338

Run the test

After you installed everything just use npm test to start the test suite.

Note

The test suite is not complete and more a proof of concept than anything else.

Filestructure

client/         # everything that will end up in the clients browser
parser/         # the parse (will probably be its own repository some day)
server/         # the backend (except for the parser)
app.js          # the starting point for this app
