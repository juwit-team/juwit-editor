Using the doT.js templating engine the parser is easily extendable and robust. All the templates itself can be found in the github repository.

Basic structure

To convert a html tag to latex code the template has to follow a few rules.

  1. the tag has to be declared in the configuration
  2. a file tag_name.dot has to be present in the matching templates folder
  3. inside the template has to be a {{= it.content }}

Example templates

ordered list (ol)

\begin{enumerate}
  % every child tag will be part of `it.content`
  {{= it.content }}
\end{enumerate}

list item (li)

\item {{= it.content }}
